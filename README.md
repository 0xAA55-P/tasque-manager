# Tasque-manager
 - Criei este projeto para praticar consumo de APIs e Handling de JSON.

## APIs

- **Bored API** - Receber tarefas ([site](https://bored-api.appbrewery.com/))

# Pré-requisitos
- Antes de começar, certifique-se de que cumpre os seguintes requisitos:
	- Python >= 3.9 (3.11+ recomendado)
	- rich, requests >= Vá para a area de Instalação

## Instalação

- Veja como instalar:

```bash
# Clonar o repositório
git clone https://github.com/0xAA55-P/tasque-manager

# Instalar dependências
pip install -r requisitos.txt

# Executar o aplicativo
python run.py
```
